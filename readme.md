# Curiosity — Synthetic Cabinet

This project is a collective experiment created during a workshop on AI and immersive media.

Each participant contributes an image and a short description.  
These fragments are then interpreted by an automated system to generate 3D objects.

The result is a shared virtual cabinet of curiosities — a speculative museum made of reconstructed artifacts from a world that never existed.

The process embraces approximation, distortion, and interpretation.  
What you see is not the object itself, but a hypothesis.
